# プロジェクト名

HOSP移行のためのGoogleカレンダー残存情報抽出

## 処理概要

Googleカレンダーのマイカレンダーにて購読しているカレンダーの情報を抽出し、スプレッドシートに出力する処理です。

## .clasp.jsonサンプル

```
{
  "scriptId": "スクリプトのID",
  "rootDir": "dist",
  "parentId": "スプレッドシートのID",
  "scriptExtensions": [
    ".js",
    ".gs"
  ],
  "htmlExtensions": [
    ".html"
  ],
  "jsonExtensions": [
    ".json"
  ],
  "filePushOrder": [],
  "skipSubdirectories": false
}
```

## 使い方

1. Googleスプレッドシートを新規作成し、`.clasp-dev.json`にスクリプトIDとスプレッドシートIDを記載します。
2. `npm run deploy`でスクリプトをプッシュし、GASで実行します。

## 📌 提供される関数（Google Apps Script）

本プロジェクトは、Google カレンダーの予定情報を取得し、  
スプレッドシートへ一覧出力・集計するための Apps Script です。

---

### 🔹 `summaryEvents()`

イベント一覧シート（`イベント一覧`）に出力されたデータをもとに、  
集計・サマリー処理を実行します。

- 内部で `summaryEvents_()` を呼び出します
- 集計ロジックの詳細は `summaryEvents.ts` を参照してください

**用途**

- 予定データの集計
- 日別・会議室別などのサマリー作成

---

### 🔹 `listEvents()`

対象カレンダーの予定を取得し、  
スプレッドシートの `イベント一覧` シートに出力します。

#### 処理内容

1. `TARGET_CALENDAR_IDS_SHEET_NAME` シートから  
   対象カレンダー ID 一覧を取得
2. 「除外」と指定されたカレンダー ID を除外
3. 祝日・テスト用などの固定除外カレンダーを除外
4. 各カレンダーのイベントを取得
5. 結果を `イベント一覧` シートに出力

#### 出力列

| 列名      | 内容                    |
| --------- | ----------------------- |
| id        | カレンダー名 / 会議室名 |
| eventId   | イベントID              |
| title     | イベントタイトル        |
| startDay  | 開始日                  |
| startTime | 開始時刻                |
| endDay    | 終了日                  |
| endTime   | 終了時刻                |
| guests    | 参加者                  |

#### 補足

- 会議室カレンダー（ID が `RESOURCE_CALENDAR_SUFFIX` で終わるもの）は  
  シートに記載された会議室名に変換して出力されます。
- 既存の `イベント一覧` シートの内容は実行時にクリアされます。

---

### 🔹 `listCalendars()`

実行ユーザーがアクセス可能な  
マイカレンダー配下のカレンダー一覧を取得し、  
`TARGET_CALENDAR_IDS_SHEET_NAME` シートに出力します。

#### 出力内容

| 列  | 内容                                 |
| --- | ------------------------------------ |
| A   | カレンダー名                         |
| B   | カレンダー ID                        |
| C   | 処理対象外にする場合は「除外」と記載 |

#### 用途

- 対象カレンダー ID 一覧の初期作成
- 処理対象カレンダーの選定
- 不要なカレンダーの除外指定

---

## ⚙️ 設定シート

### `TARGET_CALENDAR_IDS_SHEET_NAME`

対象とするカレンダー ID を管理するシートです。
ここに記載されているIDのカレンダーが、マイカレンダーに登録されていない場合はエラーで異常終了します。

| 名前           | ID                                  | 処理対象外にする場合は"除外"と記載 |
| -------------- | ----------------------------------- | ---------------------------------- |
| 会議室A        | example-room-a@resource.calendar... |                                    |
| 個人カレンダー | example@gmail.com                   | 除外                               |

- 3 列目に `除外` と記載された行は処理対象外になります。

---

## ▶️ 実行方法

1. `listCalendars()` を実行  
   → カレンダー一覧を取得してシートに出力

2. 必要なカレンダーを残し、不要なものは 3 列目に `除外` と記載

3. `listEvents()` を実行  
   → 予定一覧を `イベント一覧` シートに出力

4. `summaryEvents()` を実行  
   → イベント一覧をもとに集計処理を実行

---

## 📝 補足

- 本スクリプトは Google Apps Script 上での実行を前提としています
- 初回実行時は Google カレンダーおよびスプレッドシートへの  
  アクセス権限の承認が必要です
