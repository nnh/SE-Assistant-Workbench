# Google Drive 権限一括エクスポートツール (TypeScript版)

Google ドライブの指定フォルダ配下を再帰的に走査し、ファイル・フォルダの共有権限、オーナー、ショートカット情報などをスプレッドシートへ抽出するツールです。

## 🚀 主な機能

- **再帰的スキャン**: 指定したルートフォルダ配下の全階層を自動走査。
- **権限の可視化**: オーナー、編集者、閲覧者、アクセス種別（リンク共有状態など）を一覧化。
- **ショートカット解析**: ショートカットファイルのリンク先（ターゲットID）を特定。
- **高度な除外設定**: 「対象外フォルダ」シートにIDを記入することで、巨大なディレクトリのスキャンをスキップ。
- **レジューム機能**: 「検索済み」シートに処理済みIDを記録。中断しても続きから再開可能。
- **堅牢な設計**: TypeScriptによる型定義と、`safeGet_` によるエラーハンドリングを実装。

## 📁 プロジェクト構成

Aside（clasp）環境での開発に最適化された構成です。

- `src/configService.ts`: 初期設定やプロパティ操作。
- `src/consts.ts`: シート名やプロパティキーなどの定数管理。
- `src/driveRepository.ts`: ドライブからのデータ抽出ロジック。
- `src/index.ts`: エントリーポイント（GASから実行する関数）。
- `src/permissionService.ts`: 再帰探索やバッチ処理のメインロジック。
- `src/setup.ts`: スプレッドシートのシートセットアップロジック。
- `src/utils.ts`: 共通ユーティリティ（エラーラップ関数など）。

## 📋 事前準備

### プロジェクトのデプロイ

- Googleドライブの任意のフォルダに新規スプレッドシートを作成してください。
- プロジェクトのルートフォルダに`.clasp-dev.json`と`.clasp-prod.json`を作成してください。

```
{
  "scriptId": "スクリプトIDをここに入れる",
  "rootDir": "dist",
  "parentId": "スプレッドシートのIDをここに入れる",
  "scriptExtensions": [
    ".js",
    ".gs"
  ],
  "htmlExtensions": [
    ".html"
  ],
  "jsonExtensions": [
    ".json"
  ],
  "filePushOrder": [],
  "skipSubdirectories": false
}
```

- ターミナルでプロジェクトのルートまで移動してください。`npm run deploy`で`.clasp-dev.json`に設定したスプレッドシートにスクリプト群がプッシュされます。

### 初期セットアップ

プロジェクトをデプロイ後、GASエディタまたはメニューから `setup` 関数を実行してください。以下のシートが自動生成されます。

| シート名           | 役割                                                            |
| :----------------- | :-------------------------------------------------------------- |
| **共有権限**       | 抽出結果が出力されるメインシート。                              |
| **検索済み**       | 処理済みIDが記録されます。途中で止まった際の再開に使用します。  |
| **対象外フォルダ** | **A列**にスキャンをスキップしたいフォルダIDを入力してください。 |

### 2. スクリプトプロパティの設定

Google Apps Script の「プロジェクトの設定」にて以下のプロパティを設定してください。

- `TARGET_ROOT_FOLDER_ID`: 走査を開始するフォルダのID。

## 📖 使い方

1.  **除外設定**: スプレッドシートの「対象外フォルダ」シートに、スキャン不要なフォルダIDを登録します。
2.  **実行**: `main` 関数を実行します。
3.  **確認**: 実行ログに `✅ ◯件処理完了` と表示され、スプレッドシートにデータが蓄積されます。
4.  **再開**: タイムアウト等で終了した場合は、そのまま再度 `main` を実行すれば「検索済み」シートにない項目から自動的に再開します。

## 📊 出力項目の詳細

| 項目                   | 説明                                                     |
| :--------------------- | :------------------------------------------------------- |
| **タイプ**             | 「フォルダ」または「ファイル」                           |
| **パス**               | ルートフォルダからの相対パス                             |
| **名前**               | アイテムの名称                                           |
| **ID / URL**           | Googleドライブ上の固有IDと直リンク                       |
| **アクセス種別**       | リンク共有設定（ANYONE, DOMAIN, LIMITEDなど）            |
| **オーナー**           | 所有者のメールアドレス                                   |
| **編集者 / 閲覧者**    | 各権限を持つユーザー一覧（改行区切り）                   |
| **ショートカット元ID** | ショートカットの場合、その参照先（ターゲット）のIDを表示 |

## ⚠️ 注意事項

- **実行制限**: GASの30分制限を超えると停止します。その場合は再度実行してください。
- **権限不足**: 権限の取得に失敗した場合、失敗した項目は「!取得不可!」と表示されます。
- **シートのクリア**: 新しくスキャンを最初からやり直したい場合は、「共有権限」と「検索済み」シートのデータを削除してから実行してください。
