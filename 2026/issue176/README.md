# issue176

## 概要

Box外部共有フォームをGoogleからMicrosoftに移行するための作業用スクリプトです。
2つの外部スプレッドシートからデータを取り込み、整形・結合してアクティブスプレッドシートに出力します。

## 処理一覧

| 関数名           | 説明                                                 |
| ---------------- | ---------------------------------------------------- |
| `initProperties` | スクリプトプロパティにダミー値を登録する初期設定     |
| `importData`     | 外部ファイルからデータを取り込み、整形して出力する   |
| `joinData`       | 「フォームの回答 2」と「外部一覧」を結合して出力する |
| `createCopyInfo` | 「結合結果」シートからコピー用情報シートを生成する   |

## セットアップ

### 1. スクリプトプロパティの設定

`initProperties` を一度実行してダミー値を登録し、スクリプトエディタの「プロジェクトの設定 > スクリプトプロパティ」から実際の値に書き換えてください。

| プロパティ名     | 説明                                                                  |
| ---------------- | --------------------------------------------------------------------- |
| `INPUT_FILE1_ID` | 入力ファイル1（「フォームの回答 2」シートを含むスプレッドシート）のID |
| `INPUT_FILE2_ID` | 入力ファイル2（「外部一覧」シートを含むスプレッドシート）のID         |

### 2. アクティブスプレッドシートに必要なシート

以下のシートをあらかじめ用意してください（データを入れておく必要があるもの）。

| シート名               | 用途                                                                                                 |
| ---------------------- | ---------------------------------------------------------------------------------------------------- |
| `ディレクトリ対応表`   | L列の置換ルール。A列：置換前、B列：置換後（1行目は見出し）                                           |
| `メールアドレス対応表` | J列の置換ルール。A列：置換前、B列：置換後（1行目は見出し）                                           |
| `分割対応表`           | L列が一致する行を2行に分割するルール。A列：分割対象、B列：1行目の値、C列：2行目の値（1行目は見出し） |

以下のシートはスクリプトが自動で作成・クリアします。

| シート名           | 生成元           |
| ------------------ | ---------------- |
| `フォームの回答 2` | 入力ファイル1    |
| `外部一覧`         | 入力ファイル2    |
| `結合結果`         | `joinData`       |
| `未結合`           | `joinData`       |
| `コピー用情報`     | `createCopyInfo` |

## 実行順序

```
1. initProperties  （初回のみ）
        ↓
2. importData      （外部データの取込・整形）
        ↓
3. joinData        （フォームの回答 2 × 外部一覧の結合）
        ↓
4. createCopyInfo  （コピー用情報の生成）
```

## importData の処理詳細

1. スクリプトプロパティからファイルIDを取得して外部スプレッドシートを開く
2. 「フォームの回答 2」シートから P列に「削除」を含まない行を抽出（見出し行は必ず出力）
3. L列が分割対応表に一致する行を2行に分割
4. J列（メールアドレス）をメールアドレス対応表に従って置換
5. L列（ディレクトリパス）を以下の順で整形
   - ディレクトリ対応表に従って置換（セル内改行は `\n` に正規化）
   - `\` を `/` に統一
   - 先頭の `/Box/` または `Box/` を削除（大文字小文字不問）
   - `"` を削除
   - `box.com` の URL を削除
   - セル内の空行を削除
   - 末尾が `/` でなければ `/` を追加
6. 「外部一覧」シートをそのままコピー

## joinData の処理詳細

- 結合キー：`フォームの回答 2.L列 == 外部一覧.A列` **かつ** `フォームの回答 2.J列 == 外部一覧.D列`
- **結合結果シート**：一致した行（フォームの回答 2 の全列 + 外部一覧の全列）
- **未結合シート**：一致しなかった行（「元シート」列付きで出力）

## createCopyInfo の列マッピング

| コピー用情報列 | 結合結果シート列 |
| -------------- | ---------------- |
| A, B, C        | （空白）         |
| D, G           | B列              |
| E, F           | C列 + D列を結合  |
| H              | E列              |
| I              | L列              |
| J              | G列 + H列を結合  |
| K              | I列              |
| L              | J列              |
| M              | F列              |
| N              | K列              |
| O              | M列              |
| P              | O列              |
| Q              | N列              |
